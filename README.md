# PCD I

## Project Description
This project is a CGPA calculator designed for students and administrators. It provides functionality for managing student data, calculating GPAs, and displaying results in a user-friendly manner.

## Features
- **Student Mode**: Allows students to view their GPA and CGPA.
- **Admin Mode**: Enables administrators to enter and manage student results.
- **Welcome Interface**: A visually appealing welcome screen.

## How to Run
1. Compile the source code using a C compiler (e.g., GCC).
2. Run the compiled executable.
3. Follow the on-screen instructions to navigate through the menu.
